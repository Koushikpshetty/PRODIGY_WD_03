# PRODIGY_WD_03
Tic-Tac-Toe Web Application


Project Description

This project is a Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. It allows two players to play the classic game in a browser with real-time interaction and result detection.


Features

-Interactive 3x3 game board

-Two-player turn-based gameplay (X and O)

-Automatic win detection

-Draw detection

-Game reset functionality

-Responsive and user-friendly UI


-Technologies Used

-HTML

-CSS

-JavaScript
